# Documentation

Quick links for working with image-view.

## Install

Homebrew (macOS/Linux):

```bash
brew tap nikolareljin/tap
brew install image-view
```

Cargo (cross-platform):

```bash
cargo install --git https://github.com/nikolareljin/image-view --bin image-view
```
